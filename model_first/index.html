    Módulo para Criação e Treinamento do Modelo de Machine Learning

    Este módulo é responsável por carregar as configurações, processar os dados, treinar um modelo de árvore de decisão utilizando GridSearchCV para otimização de hiperparâmetros, e salvar o modelo treinado.

    Funções

    carregar_config

    def carregar_config(config_path: str) -> dict:
    """
    Carrega as configurações do arquivo YAML.

    Esta função lê um arquivo YAML de configuração e retorna o conteúdo como um
    dicionário Python. As configurações carregadas incluem parâmetros necessários
    para a geração de dados, treinamento do modelo e configurações de logging.

    Args:
        config_path (str): Caminho para o arquivo de configuração YAML.

    Returns:
        dict: Dicionário com as configurações carregadas.
    """

    Descrição:
    Carrega as configurações necessárias a partir de um arquivo YAML.

    Parâmetros: - config_path (str): Caminho para o arquivo de configuração YAML.

    Retorno: - dict: Dicionário contendo as configurações carregadas.

    Exemplo de Uso:

    config = carregar_config("config/config.yaml")

    carregar_dados

    def carregar_dados(nome_arquivo: str) -> pd.DataFrame:
    """
    Carrega o dataset de um arquivo CSV.

    Esta função carrega os dados de um arquivo CSV e os retorna como um DataFrame
    do pandas. É usado tanto para carregar dados de treino quanto de teste.

    Args:
        nome_arquivo (str): Nome do arquivo CSV.

    Returns:
        pd.DataFrame: DataFrame contendo o dataset carregado.
    """

    Descrição:
    Carrega os dados de um arquivo CSV para um DataFrame.

    Parâmetros: - nome_arquivo (str): Nome do arquivo CSV a ser carregado.

    Retorno: - pd.DataFrame: DataFrame contendo os dados carregados.

    Exemplo de Uso:

    dados = carregar_dados("data/dataset_treino.csv")

    preprocessar_dados

    def preprocessar_dados(dados: pd.DataFrame):
    """
    Pré-processa os dados do dataset.

    Esta função realiza o pré-processamento dos dados, separando as features (X)
    da variável alvo (y), e aplicando transformações como escalonamento das features
    numéricas e codificação one-hot das features categóricas. O pré-processador resultante
    é retornado junto com os dados processados.

    Args:
        dados (pd.DataFrame): DataFrame contendo o dataset.

    Returns:
        X (pd.DataFrame): Features do dataset pré-processadas.
        y (pd.Series): Variável alvo (preço).
        preprocessor (ColumnTransformer): Pipeline de pré-processamento.
    """

    Descrição:
    Pré-processa os dados, separando as features (X) da variável alvo (y), e aplica transformações necessárias como escalonamento e codificação one-hot.

    Parâmetros: - dados (pd.DataFrame): DataFrame contendo o dataset a ser pré-processado.

    Retorno: - X (pd.DataFrame): Features do dataset após o pré-processamento. - y (pd.Series): Variável alvo. - preprocessor (ColumnTransformer): Pipeline de pré-processamento.

    Exemplo de Uso:

    X, y, preprocessor = preprocessar_dados(dados)

    treinar_modelo

    def treinar_modelo(X, y, preprocessor, save_path: str) -> Pipeline:
    """
    Treina o modelo de árvore de decisão utilizando GridSearchCV para otimização de hiperparâmetros.

    Esta função configura um pipeline que combina o pré-processamento dos dados com
    o modelo de árvore de decisão. Em seguida, realiza a busca pelos melhores hiperparâmetros
    utilizando o GridSearchCV. Após encontrar o melhor modelo, ele é salvo no caminho especificado.

    Args:
        X (pd.DataFrame): Features do dataset.
        y (pd.Series): Variável alvo (preço).
        preprocessor (ColumnTransformer): Pipeline de pré-processamento.
        save_path (str): Caminho para salvar o modelo treinado.

    Returns:
        Pipeline: Modelo treinado com os melhores hiperparâmetros.
    """

    Descrição:
    Treina um modelo de árvore de decisão utilizando GridSearchCV para otimização de hiperparâmetros. O modelo final é salvo no caminho especificado.

    Parâmetros: - X (pd.DataFrame): Features do dataset. - y (pd.Series): Variável alvo. - preprocessor (ColumnTransformer): Pipeline de pré-processamento dos dados. - save_path (str): Caminho para salvar o modelo treinado.

    Retorno: - Pipeline: Pipeline contendo o melhor modelo treinado.

    Exemplo de Uso:

    modelo = treinar_modelo(X, y, preprocessor, "models/melhor_modelo.joblib")

    obter_nomes_features

    def obter_nomes_features(preprocessor, X) -> list:
    """
    Ajusta o pré-processador e obtém os nomes das features após o pré-processamento.

    Esta função ajusta o pré-processador aos dados de entrada e extrai os nomes
    das features que resultam do pré-processamento, incluindo tanto as features numéricas
    quanto as features categóricas codificadas.

    Args:
        preprocessor (ColumnTransformer): Pipeline de pré-processamento.
        X (pd.DataFrame): DataFrame de features.

    Returns:
        list: Lista com os nomes das features após o pré-processamento.
    """

    Descrição:
    Ajusta o pré-processador aos dados e retorna uma lista com os nomes das features resultantes, incluindo features categóricas codificadas.

    Parâmetros: - preprocessor (ColumnTransformer): Pipeline de pré-processamento. - X (pd.DataFrame): Features que serão ajustadas no pré-processador.

    Retorno: - list: Lista com os nomes das features após o pré-processamento.

    Exemplo de Uso:

    nomes_features = obter_nomes_features(preprocessor, X)

    Exemplo Completo de Uso do Módulo

    if __name__ == "__main__":
    # Carregar configurações do arquivo YAML
    config = carregar_config("config/config.yaml")

    # Configurar logging com base nas configurações carregadas
    logging.basicConfig(filename=config['logging']['log_file'], level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')

    # Carregar e pré-processar os dados de treino
    dados_treino = carregar_dados("data/dataset_treino.csv")
    X_treino, y_treino, preprocessor = preprocessar_dados(dados_treino)

    # Ajustar o pré-processador e obter os nomes das features
    feature_names = obter_nomes_features(preprocessor, X_treino)

    # Treinar o modelo e salvar o melhor modelo encontrado
    modelo = treinar_modelo(X_treino, y_treino, preprocessor, config['model']['save_path'])
    logging.info(f"Nomes das features: {feature_names}")

    Descrição:
    Este bloco de código demonstra o fluxo completo desde o carregamento das configurações, pré-processamento dos dados, até o treinamento e salvamento do modelo.

    Considerações

    • Otimização: O uso de GridSearchCV permite a otimização dos hiperparâmetros do modelo de árvore de decisão.
    • Configuração: Todas as configurações necessárias são carregadas de um arquivo YAML, permitindo flexibilidade.
    • Logging: O módulo registra as etapas importantes do fluxo, facilitando o rastreamento e a depuração.
