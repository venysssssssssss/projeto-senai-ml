    Módulo para Geração dos Datasets de Treino e Teste

    Este módulo utiliza a biblioteca Faker para gerar datasets de treino e teste com dados fictícios de barcos. A configuração do módulo é feita através de um arquivo YAML.

    Funções

    carregar_config

    def carregar_config(config_path: str) -> dict:
    """
    Carrega as configurações do arquivo YAML.

    Args:
        config_path (str): Caminho para o arquivo de configuração YAML.

    Returns:
        dict: Dicionário com as configurações carregadas.
    """

    Descrição:
    Carrega e retorna as configurações a partir de um arquivo YAML.

    Parâmetros: - config_path (str): Caminho para o arquivo de configuração YAML.

    Retorno: - dict: Dicionário contendo as configurações carregadas.

    Exemplo de Uso:

    config = carregar_config("config/config.yaml")

    gerar_dataset

    def gerar_dataset(n_amostras: int) -> pd.DataFrame:
    """
    Gera um dataset falso com n_amostras amostras.

    Args:
        n_amostras (int): Número de amostras a serem geradas.

    Returns:
        pd.DataFrame: DataFrame contendo o dataset gerado.
    """

    Descrição:
    Gera um dataset fictício com o número especificado de amostras. O dataset contém informações sobre diferentes tipos de barcos.

    Parâmetros: - n_amostras (int): Número de amostras a serem geradas.

    Retorno: - pd.DataFrame: DataFrame contendo o dataset gerado.

    Exemplo de Uso:

    dataset = gerar_dataset(100)

    salvar_dataset

    def salvar_dataset(dataset: pd.DataFrame, nome_arquivo: str):
    """
    Salva o dataset em um arquivo CSV.

    Args:
        dataset (pd.DataFrame): DataFrame contendo o dataset a ser salvo.
        nome_arquivo (str): Nome do arquivo onde o dataset será salvo.
    """

    Descrição:
    Salva o dataset gerado em um arquivo CSV.

    Parâmetros: - dataset (pd.DataFrame): DataFrame contendo o dataset a ser salvo. - nome_arquivo (str): Nome do arquivo onde o dataset será salvo.

    Exemplo de Uso:

    salvar_dataset(dataset, "data/dataset_treino.csv")

    Exemplo Completo de Uso do Módulo

    if __name__ == "__main__":
    config = carregar_config("config/config.yaml")

    logging.basicConfig(filename=config['logging']['log_file'], level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')

    n_amostras = config['data']['num_samples']
    dataset_treino = gerar_dataset(n_amostras)
    dataset_teste = gerar_dataset(int(n_amostras * config['data']['test_size']))

    salvar_dataset(dataset_treino, "data/dataset_treino.csv")
    salvar_dataset(dataset_teste, "data/dataset_teste.csv")

    logging.info("Datasets de treino e teste gerados e salvos com sucesso.")

    Descrição:
    Este bloco de código representa o uso completo do módulo, onde as configurações são carregadas, os datasets de treino e teste são gerados e salvos, e uma mensagem de sucesso é registrada no log.

    Configuração YAML Exemplo

    Abaixo está um exemplo de como o arquivo config.yaml poderia ser estruturado:

    logging:
  log_file: "logs/project.log"

data:
  num_samples: 1000
  test_size: 0.2

    Considerações

    • Bibliotecas Utilizadas:
    • pandas: Para manipulação de dados e criação de DataFrames.
    • faker: Para geração de dados fictícios.
    • yaml: Para leitura das configurações em YAML.
    • logging: Para registro de logs.

    • random: Para geração de valores aleatórios.

    • Erros Potenciais:
      Certifique-se de que o caminho para o arquivo YAML esteja correto e que o arquivo seja válido para evitar erros ao carregar as configurações.
