    Módulo para Visualização do Desempenho do Modelo

    Este módulo contém funções para visualizar o desempenho de um modelo de machine learning, incluindo a comparação entre valores reais e preditos, análise de resíduos, e visualização da importância das features.

    Funções

    plot_predictions

    def plot_predictions(y_true, y_pred):
    """
    Plota os valores reais versus os valores preditos.

    Args:
        y_true (pd.Series): Valores reais.
        y_pred (pd.Series): Valores preditos.
    """

    Descrição:
    Plota uma dispersão dos valores reais contra os valores preditos pelo modelo, juntamente com uma linha de perfeição (onde y_true é igual a y_pred).

    Parâmetros: - y_true (pd.Series): Valores reais da variável alvo. - y_pred (pd.Series): Valores preditos pelo modelo.

    Exemplo de Uso:

    plot_predictions(y_test, y_pred)

    plot_residuals

    def plot_residuals(y_true, y_pred):
    """
    Plota os resíduos (erros) do modelo.

    Args:
        y_true (pd.Series): Valores reais.
        y_pred (pd.Series): Valores preditos.
    """

    Descrição:
    Plota a distribuição dos resíduos, que são as diferenças entre os valores reais e preditos. Um histograma com uma linha KDE é usado para visualizar a distribuição dos resíduos.

    Parâmetros: - y_true (pd.Series): Valores reais da variável alvo. - y_pred (pd.Series): Valores preditos pelo modelo.

    Exemplo de Uso:

    plot_residuals(y_test, y_pred)

    plot_feature_importances

    def plot_feature_importances(model, feature_names):
    """
    Plota a importância das features do modelo.

    Args:
        model (Pipeline): Modelo treinado.
        feature_names (list): Lista com os nomes das features.
    """

    Descrição:
    Plota a importância das features usadas pelo modelo de árvore de decisão, ordenando-as por sua importância.

    Parâmetros: - model (Pipeline): Pipeline do modelo treinado, que deve conter um regressor com atributo feature_importances_. - feature_names (list): Lista com os nomes das features, na mesma ordem que foram usadas para treinar o modelo.

    Exemplo de Uso:

    plot_feature_importances(model, feature_names)

    Exemplo Completo de Uso do Módulo

    # Supondo que y_test e y_pred sejam os valores reais e preditos, respectivamente
plot_predictions(y_test, y_pred)

# Plote os resíduos
plot_residuals(y_test, y_pred)

# Plote a importância das features (feature_names deve ser uma lista com os nomes das features)
plot_feature_importances(model, feature_names)

    Considerações

    • Visualização de Desempenho: As funções de plotagem fornecem uma maneira visual de entender o desempenho do modelo e identificar possíveis áreas de melhoria.
    • Importância das Features: A importância das features ajuda a identificar quais variáveis estão contribuindo mais para as previsões do modelo.
    • Personalização: As funções de plotagem são facilmente personalizáveis, permitindo ajustes nos estilos e formatos de gráficos conforme necessário.

    Dependências

    • matplotlib: Usado para a criação de gráficos e visualizações.
    • seaborn: Usado para a criação de gráficos estatísticos, como a distribuição de resíduos.
