    Configuração de Carregamento

    Aqui está a documentação em formato Markdown para a função load_config:

    Função load_config

    import yaml

def load_config(config_path):
    with open(config_path, 'r') as file:
        config = yaml.safe_load(file)
    return config

    Descrição

    A função load_config carrega e retorna o conteúdo de um arquivo de configuração no formato YAML.

    Parâmetros

    • config_path (str): Caminho para o arquivo de configuração YAML. Este parâmetro deve ser uma string que representa o caminho do arquivo que contém as configurações.

    Retorno

    • config (dict): Um dicionário Python contendo os dados carregados do arquivo YAML. Cada chave no dicionário corresponde a uma chave no arquivo YAML, e os valores correspondem aos valores associados a essas chaves.

    Exemplo de Uso

    # Exemplo de uso da função load_config

config_path = "config.yaml"
config = load_config(config_path)

# Acessando um valor específico no dicionário de configuração
api_key = config["api_key"]

    Considerações

    • Erros Potenciais: Se o arquivo especificado em config_path não existir ou não for um arquivo YAML válido, a função pode levantar exceções (FileNotFoundError, yaml.YAMLError).
    • Segurança: A função utiliza yaml.safe_load, que é uma maneira segura de carregar dados YAML, protegendo contra a execução de código arbitrário em caso de arquivos YAML maliciosos. ```
