    Módulo para Avaliação do Modelo de Machine Learning

    Este módulo é responsável pela avaliação de um modelo de machine learning utilizando métricas como erro absoluto médio (MAE), erro quadrático médio (MSE) e coeficiente de determinação (R²). Ele carrega o modelo treinado, executa a avaliação no conjunto de teste e registra as métricas obtidas.

    Funções

    carregar_config

    def carregar_config(config_path: str) -> dict:
    """
    Carrega as configurações do arquivo YAML.

    Args:
        config_path (str): Caminho para o arquivo de configuração YAML.

    Returns:
        dict: Dicionário com as configurações carregadas.
    """

    Descrição:
    Carrega as configurações necessárias do arquivo YAML especificado.

    Parâmetros: - config_path (str): Caminho para o arquivo de configuração YAML.

    Retorno: - dict: Dicionário contendo as configurações carregadas.

    Exemplo de Uso:

    config = carregar_config("config/config.yaml")

    carregar_modelo

    def carregar_modelo(caminho_modelo: str):
    """
    Carrega o modelo treinado de um arquivo.

    Args:
        caminho_modelo (str): Caminho para o arquivo do modelo.

    Returns:
        Pipeline: Modelo carregado.
    """

    Descrição:
    Carrega um modelo de machine learning previamente treinado a partir de um arquivo.

    Parâmetros: - caminho_modelo (str): Caminho para o arquivo que contém o modelo treinado.

    Retorno: - Pipeline: O modelo carregado, pronto para ser utilizado em previsões.

    Exemplo de Uso:

    modelo = carregar_modelo("models/modelo_treinado.joblib")

    avaliar_modelo

    def avaliar_modelo(modelo, X_teste, y_teste):
    """
    Avalia o modelo usando o dataset de teste.

    Args:
        modelo (Pipeline): Modelo treinado.
        X_teste (pd.DataFrame): Features do dataset de teste.
        y_teste (pd.Series): Variável alvo do dataset de teste.

    Returns:
        dict: Métricas de avaliação do modelo.
    """

    Descrição:
    Avalia o desempenho do modelo utilizando um conjunto de dados de teste e calcula as métricas de avaliação.

    Parâmetros: - modelo (Pipeline): Modelo treinado que será avaliado. - X_teste (pd.DataFrame): Conjunto de features do dataset de teste. - y_teste (pd.Series): Variável alvo correspondente ao dataset de teste.

    Retorno: - dict: Um dicionário contendo as métricas de avaliação, incluindo: - MAE: Mean Absolute Error (Erro Absoluto Médio) - MSE: Mean Squared Error (Erro Quadrático Médio) - R2: Coeficiente de Determinação (R²)

    Exemplo de Uso:

    metrics = avaliar_modelo(modelo, X_teste, y_teste)
print(metrics)

    Exemplo Completo de Uso do Módulo

    if __name__ == "__main__":
    config = carregar_config("config/config.yaml")

    logging.basicConfig(filename=config['logging']['log_file'], level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')

    dados_teste = pd.read_csv("data/dataset_teste.csv")
    X_teste, y_teste, preprocessor = preprocessar_dados(dados_teste)
    modelo = carregar_modelo(config['model']['save_path'])
    metrics = avaliar_modelo(modelo, X_teste, y_teste)
    for metric, value in metrics.items():
        logging.info(f"{metric}: {value:.4f}")

    Descrição:
    Este bloco de código demonstra o fluxo completo de carregamento de configurações, carregamento do modelo, preparação dos dados de teste, avaliação do modelo e registro das métricas obtidas no log.

    Considerações

    • Pré-processamento: O pré-processamento dos dados de teste é realizado pela função preprocessar_dados importada do módulo model_first.
    • Configuração: As configurações para o caminho do modelo e do arquivo de log são carregadas de um arquivo YAML.
    • Métricas de Avaliação: As principais métricas de avaliação incluem MAE, MSE e R², que fornecem uma visão abrangente do desempenho do modelo.
